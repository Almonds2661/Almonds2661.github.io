Java FX Server / Client Program

This was a Java program that would run a multithreaded server that would wait for incoming connections using Server Sockets. It would use Java-FX as a GUI framework for the clients while reported feedback to the server via its console. It used abstracted "packets" for network transfers and future packets types can be added later in a modular fashion.

The server would use threads to listen for packets then send those packets to the clients that would also have a threads listening for them. Clients would receive the packets and display messages. Clients could also leave and join without causing disruptions. They also used a JSON file for their name and display colors.
