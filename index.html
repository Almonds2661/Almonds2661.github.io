AI Data Import

This was a group project with a small group of students working under real world clients in Saskatoon under our instructor's supervision. The clients system used PHP as a framework, and it is what we used to serve the website. On the backend, it operated on a LAMP stack (Linux, Apache, MySQL, PHP).

The project took in PDFs, Excel sheets, or CSV data and with AI it mapped it to our clients database schema. The database utilized Doctrine, and Object Relational Model to map the data to our schema. The program would then display this data to the user before they imported it to the backend database.
